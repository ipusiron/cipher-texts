## アメリカ独立宣言（United States Declaration of Independence）の原文の入手先

アメリカ独立宣言（United States Declaration of Independence）の原文を入手しました（[https://www.archives.gov/](https://www.archives.gov/)）。
ここは、アメリカ国立公文書記録管理局（NARA: National Archives and Records Administration）の公式サイトで、米国政府が運営しています。
.govドメインは米国連邦政府機関にのみ付与されるため、偽サイトのリスクもほぼありません。

アメリカ国立公文書記録管理局のWebサイトでは、米国の歴史的文書（独立宣言、憲法など）、政府の公式記録・公文書、軍事記録・系譜調査資料、大統領図書館の資料といったものが公開されています。
歴史研究や一次資料の参照において世界的に権威あるアーカイブなので、安心して利用できます。

ここにアメリカ独立宣言の原文も保存されています。

[https://www.archives.gov/founding-docs/declaration](https://www.archives.gov/founding-docs/declaration)
